# Project Roadmap (2025-2028)

This document outlines the strategic vision for the D&D Character Sheet Manager. The roadmap is structured by semantic versioning, providing a clear path from the current application to a full-featured, collaborative platform for D&D enthusiasts.

---

### **v0.1.0 - The Foundation (Q3 2025)**

This version represents the initial public release and the core foundation of the application.

* [✓] Core Character Management: Full creation, editing, and saving of D&D 5e character sheets.
* [✓] User Authentication: Secure user registration and login system.
* [✓] Open5e Data Integration: Automated caching of data for classes, races, spells, and equipment from the Open5e API.
* [✓] AI-Powered Generation: Initial implementation of Gemini and Imagen APIs for generating character backstories and portraits.
* [✓] Responsive UI: A functional and themed interface for both desktop and mobile devices.

---

### **v0.2.0 - Gameplay Enhancements (Q4 2025)**

This release cycle focuses on features that directly improve the in-session gameplay experience.

* [ ] Interactive Dice Roller: Integrate a dice roller directly into the character sheet. Users will be able to click on an ability score, skill, or attack to automatically roll the appropriate dice.
* [ ] Active Effects and Condition Tracking: Implement a system to track conditions (e.g., "Blinded," "Poisoned") and other temporary effects. Applying a condition will automatically update the character's stats and rolls where applicable.

---

### **v0.3.0 - Deeper Character Progression (Q2 2026)**

This version will introduce more complex character management features and enhance the AI's utility.

* [ ] Automated Leveling Up: Create a "Level Up" wizard that guides users through advancing their character. It will automatically update hit points, proficiency bonuses, and prompt users to select new spells and features based on their class.
* [ ] AI-Powered Adventure Log: Add an "Adventure Log" for session notes. An integrated "AI Scribe" (using the Gemini API) can expand on notes, create narrative summaries, or generate plot hooks based on the log's content.

---

### **v0.4.0 - Dungeon Master & Creative Tools (Q4 2026)**

This release will expand the application's scope to include tools specifically for Dungeon Masters.

* [ ] NPC and Quest Generators: Leverage the Gemini API to create on-the-fly generators for Non-Player Characters (NPCs), quests, and locations.
* [ ] Character Sharing (Read-Only): Enable users to share their character sheets via a public, view-only link.

---

### **v1.0.0 - The Collaborative Platform (2027)**

The official v1.0 release will mark the transition from a solo tool to a collaborative platform for entire gaming groups. This will be a major milestone for the project.

* [ ] Campaign Management: Allow users to create "Campaigns" and invite other users to join.
* [ ] Full Character Sharing and Privacy Controls: Within a campaign, players will be able to view each other's characters, and the DM will have a special dashboard with full access.
* [ ] DM-Specific Tools: Create a suite of tools for Dungeon Masters within the campaign view, such as a campaign calendar, a shared document for notes, and an initiative tracker.

---

### **Future Vision (2028 and Beyond)**

Post-v1.0, the focus will shift to customization, expansion, and community-driven content.

* [ ] Homebrew Content Creator: Allow users to create their own custom content, such as homebrew races, classes, spells, and items, and use it in their character sheets.
* [ ] Marketplace for Community Content: A platform for users to share and discover homebrew content created by the community.
* [ ] Advanced API Integrations: Explore integrations with other TTRPG platforms and tools.
* [ ] 3D Multiplayer with AI-Generated 3D Models: Introduce a 3D multiplayer mode where players can interact with AI-generated 3D models of their characters and the game world.
